## Using the code:
Our code is stable while using Python 3.10.4, CUDA 11.8, PyTorch 2.3.0
其他需安裝的 Python Package:
```
pip install numpy
pip install opencv-python
pip install torchvision
pip install timm
pip install matplotlib
```
